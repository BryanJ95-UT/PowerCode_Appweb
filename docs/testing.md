# Testing del Sistema - Power Code

## Objetivo

Verificar el correcto funcionamiento de los módulos principales del sistema Power Code mediante pruebas funcionales, visuales y de navegación.

---

# Tipos de Pruebas Realizadas

## 1. Pruebas de Registro

Se validó:

- Registro correcto de usuarios
- Correos duplicados
- Confirmación por correo
- Campos vacíos

Resultado: Exitoso

---

## 2. Pruebas de Login

Se validó:

- Acceso con credenciales correctas
- Error con contraseña incorrecta
- Usuario no registrado
- Redirección según rol

Resultado: Exitoso

---

## 3. Pruebas CRUD

Se validó:

- Alta de clientes
- Edición de datos
- Eliminación
- Consulta general

Resultado: Exitoso

---

## 4. Pruebas Frontend

Se validó:

- Navegación entre páginas
- Menús
- Botones
- Formularios
- Responsive design

Resultado: Exitoso

---

## 5. Pruebas API

Endpoints probados:

- /api/register

Resultado: Exitoso

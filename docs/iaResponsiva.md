# Implementación de IA y Diseño Responsivo - Power Code

## Objetivo

Aplicar herramientas de inteligencia artificial como apoyo en el desarrollo del sistema Power Code, así como implementar diseño responsivo adaptable a diferentes dispositivos.

---

# Uso de Inteligencia Artificial

Durante el desarrollo se utilizaron herramientas de IA para apoyo en:

## Diseño UI/UX

- Mejora visual de interfaces
- Sugerencias de distribución
- Optimización de experiencia de usuario

## Programación

- Corrección de errores
- Mejora de código JavaScript
- Optimización de rutas backend
- Organización de estructura del proyecto

## Documentación

- Redacción técnica
- Mejora de README
- Estructuración de requerimientos

---

# Diseño Responsivo Implementado

El frontend fue adaptado mediante CSS para distintos tamaños de pantalla:

## Escritorio

Resolución completa con dashboard expandido.

## Tablet

Ajuste de columnas y menús.

## Móvil

Menús compactos, tarjetas verticales y formularios adaptados.
